## 选择原因

- 这是 RuoYi-Vue-Plus 5.X 对应的统一 UI 前端仓库，和后端框架接口/权限模型匹配。
- 使用 Vue 3、TypeScript、Vite、Pinia、Vue Router、Element Plus。
- 已具备后台管理系统的登录、权限路由、菜单、字典、系统管理等基础能力。

## PatrolLink 后续改造方向

- 将系统品牌替换为智能执法耳机指挥后台。
- 增加工作台、指挥调度视频墙、警力一张图、布控预警、设备管理、媒体证据、SOS、消息通知、统计分析页面。
- 接入高德地图 Web SDK。
- 接入后端 WebSocket，处理设备状态、位置、预警、SOS、流媒体会话状态。
- 将默认业务菜单收敛为 PatrolLink 指挥后台的信息架构。
